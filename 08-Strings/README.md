# JavaScript Strings

This folder contains beginner-friendly examples of JavaScript strings.

## 📚 Topics Covered

- Create String
- String Length
- toUpperCase()
- toLowerCase()
- trim()
- charAt()
- indexOf()
- includes()
- slice()
- replace()
- split()
- repeat()
- Template Literals
- String Concatenation
- Reverse String
- Palindrome Check
