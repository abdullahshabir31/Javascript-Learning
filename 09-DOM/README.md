# JavaScript DOM

This folder contains beginner-friendly examples of JavaScript DOM (Document Object Model) manipulation.

## 📚 Topics Covered

- getElementById()
- getElementsByClassName()
- getElementsByTagName()
- querySelector()
- querySelectorAll()
- Change Text Content
- Change HTML Content
- Change CSS Styles
- getAttribute()
- setAttribute()
- createElement()
- appendChild()
- remove()
- classList
- addEventListener()
- removeEventListener()

## 🎯 Learning Outcome

After completing this folder, you will understand how to select, modify, create, remove, and style HTML elements using JavaScript DOM methods. You will also learn how to handle user interactions with events.
