# JavaScript Objects

This folder contains beginner-friendly examples of JavaScript objects.

## 📚 Topics Covered

- Creating Objects
- Accessing Object Properties
- Updating Object Properties
- Updating Object Values
- Product Object Example
- Profile Object Example
- typeof with Objects

## 📂 Files

- 01-create-object.js
- 02-access-object-property.js
- 03-update-object-property.js
- 04-update-object-value.js
- 05-product-object.js
- 06-profile-object.js
- 07-typeof-object.js

## 🎯 Learning Outcome

After completing this folder, you will understand how to create JavaScript objects, access and update their properties, and inspect object data types using the `typeof` operator.
